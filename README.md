🛒 TechStore – Dynamic E-commerce Frontend Application

TechStore is a high-performance, fully responsive e-commerce web application built using Vanilla JavaScript, HTML5, and CSS3.
It showcases the ability to create a scalable, data-driven user interface without relying on external frameworks.

🚀 Live Demo

🔗 View Live Project: [ https://krishkumar2227.github.io/TechStore-E_Commerce_Website/]

🛠️ Features & Technical Highlights=>
⚡ Dynamic Product Rendering
Built a data-driven UI where products are dynamically generated from a JavaScript array
Simulates real-world backend/database integration
🔍 Real-Time Search
Implemented instant search functionality using .filter() and .includes()
Provides fast and responsive user feedback
🎯 Multi-Category Filtering
Designed a category-based filtering system
Supports seamless navigation across:
Audio
Wearables
Peripherals
🛒 Shopping Cart System
Developed a side-drawer cart interface
Implemented:
Real-time item count updates
Dynamic total price calculation
Used event delegation for efficient state updates
📱 Responsive Design
Built using CSS Grid & Flexbox
Optimized for:
Mobile
Tablet
Desktop
