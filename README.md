TechStore – Dynamic E-commerce Frontend Application
TechStore is a high-performance, responsive e-commerce web interface developed using Vanilla JavaScript, HTML5, and CSS3. The project demonstrates the ability to build a functional, data-driven user experience without the overhead of heavy frameworks.

🛠️ Technical Architecture & Features
Dynamic Data Rendering: Implemented a data-driven approach where the product grid is generated dynamically from a JavaScript object array, simulating a real-world database integration.

Real-time Search Algorithm: Engineered a high-speed search functionality using the .filter() and .includes() methods to provide instantaneous feedback to the user.

Multi-Criteria Filtering: Developed a category-based sorting system that works in tandem with the search bar to allow users to navigate through "Audio," "Wearables," and "Peripherals."

State-Persistent Shopping Cart: * Developed a side-drawer cart UI.

Implemented real-time calculations for item counts and total pricing.

Used event delegation to manage cart updates efficiently.

Responsive UI/UX: Utilized CSS Grid and Flexbox to ensure a seamless experience across mobile, tablet, and desktop breakpoints.
